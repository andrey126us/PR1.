## Ответы на контрольные вопросы

### 1. Понятие виджета

Виджет — это графический элемент интерфейса (кнопка, текстовое поле, изображение и т.д.), который наследуется от класса `View`. Используется для взаимодействия пользователя с приложением.

### 2. Графические примитивы

Графические примитивы — базовые фигуры для рисования: точка, линия, прямоугольник, круг, эллипс, дуга. В Android рисуются с помощью классов `Canvas` (холст) и `Paint` (кисть).

### 3. Переопределение метода

Переопределение метода — это изменение реализации метода родительского класса в дочернем классе с помощью аннотации `@Override`. Позволяет задать специфическое поведение для конкретного класса.

### 4. Наследование

Наследование — механизм ООП, при котором дочерний класс получает все свойства и методы родительского класса. В Android используется, например, при создании собственных виджетов (наследование от `View`).
